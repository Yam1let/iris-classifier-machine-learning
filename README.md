## Overview

This project implements a Machine Learning classifier using the Iris dataset.
The model is trained to classify iris flowers into three species based on their physical measurements.

The implementation uses Logistic Regression from scikit-learn and demonstrates the fundamental workflow of supervised learning.

---

## Features

* Dataset loading using scikit-learn
* Train/test data split
* Logistic Regression model
* Model evaluation using accuracy score
* Classification report with precision, recall, and F1-score

---

## Machine Learning Workflow

1. Load dataset
2. Split data into training and testing sets
3. Train model
4. Generate predictions
5. Evaluate model performance
